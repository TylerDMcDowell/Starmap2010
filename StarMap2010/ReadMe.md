# The Astraeum 2548 Codex

Formerly known as StarMap2010.

The Astraeum 2548 Codex is a canonical reference and management system
for a persistent science-fiction universe set in the year 2548 CE.

It serves as the authoritative source for star systems, planetary bodies,
political entities, infrastructure, and historical continuity.

This project is long-lived, curated, and intentionally conservative
in its technology choices.
